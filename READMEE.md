# Resume Analyzer

A web app to analyze resumes using AI, built with FastAPI (backend) and Streamlit (frontend).

## Project Structure

- `app/` - Backend FastAPI app code  
- `app_frontend.py` - Streamlit frontend app  
- `requirements.txt` - Python dependencies  

## Setup Instructions

- pip
